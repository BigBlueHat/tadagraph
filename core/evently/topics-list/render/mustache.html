Topics is how we group the conversations by subject. To open a topic, you just include its name in square brackets in the message.

For example, [1st things to do] would create a topic with such name, if it is your first message into this topic, or would link the message to this topic, if topic is already there.

Each topic can have multiple views, as it is defined by the semantic tag. For example, #todo is for topics of type "Todo List". Subject of a forum discussion and a name of the Wiki articles are also topics.

If you need to add views to the topic - you just add corresponding tag. You can do it from the Topic details page, right in the heading of the topic. By default, each Topic only gets a timeline view.
